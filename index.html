Welcome to Fingersmith.

This site is a public repo for experimentation, notes and demos associated with the use of WebRTC fingerprints as addresses.

Here are a couple of videos to get you started:

  • Yopet.us - a minimal implementation for use with your feathered or furry friends
  • Kranky Geek - Some background on how and why.

TLDR;

Using WebRTC fingerprints as addresses in a federated communication system gives easy MiTM protection and secure P2P communication without the need to trust the federating systems. (You still need to trust the javascript of course.) This technique is the subject of a pending patent.

The repo contains:

  • a minimal webRTC signalling service written in scala -Based on Socko
  • sample pages to talk with various devices and endpoints (EV3 Lego dog, Beaglebone black, etc)
  • example code to do MiTM attacks on webRTC video calls (if you control a signalling server)
  • example code to pre-exchange fingerprints out-of-band using QRcodes and proximity
